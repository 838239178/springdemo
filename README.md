# :leaves:Spring+SpringMvc+Mybatis Demo

这是一个使用Spring+SpringMvc+Mybatis完成的一个简单的后端小项目，主要是对一阶段学习的总结和实践。

## 数据库支持

- 数据库版本：SQLServer2019 v18.8
- 数据连接池：com.alibaba.druid v1.2.4

## 功能

后台返回类型皆为JSON数据

- 用户登录管理
- 数据库增删查改
- 统一完善的异常处理
- 切面增强日志系统
- 权限检验拦截器
